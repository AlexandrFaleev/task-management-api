# Tasks Management System API
API системы управления задачами.

## О проекте
Проект написан на NestJS с использованием PostgreSQL БД и Prisma.
В API реализованы 2 основные роли: ADMIN и USER.

### Возможности для пользователя с ролью ADMIN
Пользователь с ролью ADMIN может:
1. Смотреть список всех пользователей;
2. Смотреть список всех задач;
3. Удалять любую задачу.

### Возможности для пользователя с ролью USER
Пользователь с ролью ADMIN может:
1. Создавать задачи;
2. Посмотреть список своих задач;
3. Посмотреть информацию о конкретной задаче по её ID;
4. Редактировать задачу;
5. Удалять любую задачу.
